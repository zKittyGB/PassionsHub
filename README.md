# 🎨 Hobbies Community – React Application

Hobbies Community is a web application that allows users to share their passions, hobbies, and personal projects.  
Users can create an account, publish hobby posts, browse categories, interact with others, and explore a community-driven feed.

This project is built with **React + Vite** and focuses on clean UI, reusable components, and modern frontend practices.

---

## 🚀 Features

### 🔐 Authentication  
- User registration and login  
- Session handling

### 📝 Hobby Posts  
- Create, edit, and delete your own hobby posts  
- Add category, description, tags, and images  
- Display hobby cards in a responsive layout  
- Like system

### 🔎 Search & Filtering  
- Search posts by keywords  
- Filter by category  
- Filter by likes

### 👤 User Profiles  
- Public profile pages with user-specific posts  

---

## 🛠️ Tech Stack

### Frontend
- **React**
- **Vite**
- **React Router**
- **Axios / Fetch API**
- **Custom CSS (no Tailwind, no UI framework)**

### Backend
The backend will provide:
- Authentication system  
- CRUD operations for users and posts
- Image upload support  

It will be implemented in **PHP + MySQL**.
